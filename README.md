# 📰 Detector de Fake News utilizando Machine Learning

## 📌 Sobre o projeto

Este projeto tem como objetivo desenvolver um modelo de Machine Learning capaz de classificar notícias como verdadeiras ou falsas utilizando técnicas de Processamento de Linguagem Natural (NLP).

O projeto contempla todas as etapas de um fluxo de Ciência de Dados:

- Coleta dos dados
- Limpeza do texto
- Pré-processamento
- Vetorização utilizando TF-IDF
- Treinamento dos modelos
- Avaliação dos resultados

---

## Tecnologias utilizadas

- Python
- Pandas
- NumPy
- Scikit-Learn
- NLTK
- Matplotlib
- Seaborn

---

## Técnicas utilizadas

- Processamento de Linguagem Natural (NLP)
- TF-IDF
- Tokenização
- Remoção de StopWords
- LinearSVC
- Avaliação por Accuracy
- Matriz de Confusão

---

## Fluxo do projeto

1. Importação da base de dados
2. Tratamento dos textos
3. Vetorização utilizando TF-IDF
4. Separação entre treino e teste
5. Treinamento do modelo
6. Avaliação do desempenho

---

## Resultados

O modelo apresentou elevada acurácia na classificação das notícias, demonstrando que técnicas clássicas de NLP combinadas com algoritmos lineares continuam sendo extremamente eficientes para problemas de classificação textual.
